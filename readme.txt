This is a simple sidebar share widget. Allowing you to add a number of different social media share buttons to your blogs sidebar.

== Description ==

I created this widget as I was unable to find a simple widget which would allow me to put a number of different social media icons on my Wordpress Blog. 

This allows your users to share any page of your blog on their chosen social network site. 

== Installation ==

1. Upload the folder `simple-sidebar-share` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the widget to your sidebar through the WordPress widget menu
1. Use the inbuilt settings for the widget to add and remove any buttons that you do or don't want


== Frequently Asked Questions ==

= How do you choose multiple social networks? =

Press and hold select whilst choosing multiple different social networks. You can choose as many as you want to add.
